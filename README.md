## 📌 О проекте

Сайт представляет собой адаптивную одностраничную визитку, которая:

- Рассказывает обо мне как о специалисте.
- Демонстрирует мои навыки и инструменты (с визуальными прогресс-барами).
- Показывает все услуги: разработка лендингов, веб-приложений, автоматизация, AI-решения.
- Содержит портфолио с реальными кейсами и ссылками на описания.
- Позволяет быстро связаться со мной через форму или социальные сети.

## 🛠 Технологии

- **Вёрстка:** HTML5, CSS3 (чистый код, без фреймворков)
- **Стилизация:** CSS-переменные, адаптивная сетка, flexbox, grid
- **Интерактивность:** Vanilla JavaScript (плавный скролл, модальное окно, мобильное меню)
- **Хостинг:** GitHub Pages
- **Дополнительно:** файлы услуг в формате `.docx` для скачивания
